To create a new bucket
aws s3 mb s3://tamilcloud

To upload the file to s3 bucket
aws s3 cp demo.txt s3://tamilcloud

To download file from s3 bucket
aws s3 cp s3://tamilcloud/demo.txt ./

To delete s3 bucket file
aws s3 rm s3://tamilcloud/demo.txt

To Sync the local directory with s3 bucket 

aws s3 sync s3upload s3://tamilcloud/s3upload

aws s3 sync s3://tamilcloud/s3upload s3upload

To see s3 bucket content
aws s3 ls s3://tamilcloud/s3upload
